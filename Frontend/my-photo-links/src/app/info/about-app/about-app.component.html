ABOUT THE APP

My Photo Links is a one stop solution to access all your photos/drive links anytime anywhere. The idea of developing the app came from the difficulty faced by everyone in searching the photos of an event/trip when needed.


Features :-

  • Individual user account by signup & login.
  • Forgot password option to recover password.
  • Search photo links with filter - keywords/event date range/tags.
  • Display photo links based on the user tags so that user cant view the photos he/she is not supposed to.
  • Add new photo links and assign tags to the link.
  • Edit/delete existing links.
  • View & edit our own user profile.
  • Give feedback - User can submit feedback, report issues and give suggestions.
  • View feedbacks - Track all the feedbacks and respond accordingly.
  • Manage users - Block a user, Assign/remove admin role to user, modify tags assigned to a user.
  • Manage tags - Add new tags, edit existing tag and delete a tag.
  • Color indications - Making use of CSS, system will display users with admin role, blocked users, users with no tags, editable/non-editable photo links, unresolved issues reported by users and many more so that user will have a clear view.

  • Role based authorization :

    • USER : Search and view photo links
    • ADMIN : Add, Edit and Delete photo links
    • SUPERADMIN : Manage users and tags, view feedbacks and respond
More features coming soon....

Technologies :-

  • Backend : Java 8, Spring boot - rest - jpa, Spring security - JWT
  • Database : My SQL (GCP Cloud SQL)
  • Frontend : Angular, Angular Material, HTML5, CSS3, Flex layout
  • 3rd party libraries : SweetAlert2, NGX UI Loader, Animate.CSS
  • Hosting : Google Cloud Platform - App engine
